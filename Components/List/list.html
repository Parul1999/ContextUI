List

List is used to display items in chronological order.

• Simple List

A simple list is a list which could be nested/unordered or ordered.

  • Clothing
  • Electronics
  • Kitchen
    1. Milk
    2. Bread
    3. Cheese
  • Music
  • Posters
            
     <ul class="simple-list">
         <li class="simple-list-item">Clothing </li>
         <li class="simple-list-item">Electronics</li>
         <li class="simple-list-item">Kitchen
             <ol class="simple-nested-list">
                 <li>Milk</li>
                 <li>Bread</li>
                 <li>Cheese</li>
             </ol>
         </li>
         <li class="simple-list-item">Music</li>
         <li class="simple-list-item">Posters</li>
     </ul>
