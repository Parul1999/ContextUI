Input

Input is used to take data from the user.

Examples :

• Input

There are different types of input , a few as mentioned below . To apply the styling to any of those input following are the things to do:

  1. Put label after the input tag
  2. Enclose the label and input tag in a div with a class of field

          
 <div class="field">
   <input class="input" type="password" name="password" id="password" placeholder="*" autocomplete="off" />
   <label class="label" for="password">Password</label>
 </div>

• Input with validations

Input can be a valid or invalid input as per the conditions If an input is valid add class valid-input to input tag and valid-label to the label tag. Similarly invalid-input and invalid-label to the invalid input 

          
 <div class="field ">
   <input class="invalid-input input" type="emailId" name="emailId" 
   id="emailId" placeholder="parul@example.com"
     autocomplete="off" />
   <label class="label invalid-label" for="emailId">Email</label>
 </div>
