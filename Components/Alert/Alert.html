Alert

Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.

Examples

The Alert component can have a heading , sub-heading and an icon too.
To make an alert use class alert-container and alert-(varient).The varients are :

  1. info
  2. error
  3. warning
  4. success

The icons used in the alert are from the google-icons as mentioned in the introduction. The icons used in the alert i.e mentioned in the span can be changed as :

  1. info : info
  2. error : error
  3. warning : warning_amber
  4. success : task_alt

You can choose icons of your choice as well and just replace the name in the span or replace the span itself.
