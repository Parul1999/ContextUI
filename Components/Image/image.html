Images

Cards contain content and actions about a single subject.

Examples :

• Round Image

Round Image is a circular image, with a default dimension of 200px X 200px.

          
 <img src="path-of-image" alt="round-image" class="round-img" />
open_in_full content_copy

• Square Image

Square Image is a image in square shape, with a default dimension of 200px X 200px.

          
  <img src="path-of-image" alt="round-image" class="sq-img" />
open_in_full content_copy

• Responsive Image

Any image is responsive , whose size changes according to the screen window. It can be checked by resizing the window. Any image can be made responsive by adding class res-img 

          
<img src="path-of-image" alt="responsive-image" class="res-img" />
