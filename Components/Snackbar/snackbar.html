Snackbar

Snackbar act as popup/toast/confirmation when clicked on any

Examples

There are three types of snackbar :

  1. Baseline : snackbar-baseline
  2. Leading : snackbar-leading
  3. Stacked : snackbar-stacked

These classes can be added along with snackbar-btn to create the button.
