Buttons

A Button triggers an action or an event.

• Basic Buttons

Basic buttons can be of two types, as listed below.

  1. Primary : btn-primary
  2. Secondary : btn-secondary

      
  <button class="btn btn-primary">Primary</button>
  <button class="btn btn-secondary">Secondary</button>

• Link Button

A button can be made a link button by adding btn-link and wrapping text around anchor tag. 

          
  <button class="btn btn-link">
  <a href="link-to-redirect">Click Here</a>
  <span class="material-icons-outlined">
    open_in_new
  </span>
  </button>

• Outline Buttons

There are four types of outline, and to implement the same the following class-names are :

  1. Danger : btn-out-danger
  2. Success : btn-out-success
  3. Info : btn-out-info
  4. Warning : btn-out-warning

      
  <button class="btn btn-out-danger">Danger</button>

  2. Success : btn-sol-success
  3. Info : btn-sol-info

• Text Buttons

A text button is a button which trigger action/event but appears like a text. To make a button text-button , following classes are added

  1. Danger : btn-txt-danger
  2. Primary : btn-txt-primary
  3. Info : btn-txt-info
  4. Warning : btn-txt-warning

          
  <button class="btn btn-txt-danger">Danger</button>

• Floating Action Button

A floating action button (FAB) performs the primary, or most common, action on a screen. It appears in front of all screen content.

Any button can be made a floating action button by adding fab-icon 

          
  <button class="btn fab-icon">
   <span class="material-icons-outlined"> navigation </span>Navigation
  </button>
