Sharky is the UI framework, which helps Developers to use best styles and components,
by using pre-defined classnames. It serves as a framework, and a lot of its visual design.

In order to link components of Sharky to your project, Copy the Link given below and paste it in the head tag of your html, above all other style sheets. 

        
<link rel="stylesheet" href="https://sharkyui.netlify.app/css/component.css"/>

In order to link a specific component, change the link below and paste it in the head tag of your html, above all other style sheets. 


    
<link rel="stylesheet" href="https://sharkyui.netlify.app/Components/Varient_Name/Varient_Name.css"/>

Examples : 
1.To intoduce alert CSS
<link rel="stylesheet" href="https://sharkyui.netlify.app/Components/alert/alert.css"/>

2.To introduce text-utilities CSS
<link rel="stylesheet" href="https://sharkyui.netlify.app/Components/textutilities/textutilities.css"/>
